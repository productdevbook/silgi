## Guards vs Wraps

| | Guard | Wrap |
|---|---|---|
| **Pattern** | Return context additions | Onion with `next()` |
| **Speed** | ~47 ns | ~150 ns |
| **Sync** | Yes | No (always async) |
| **Use case** | Auth, enrichment, validation | Timing, logging, error capture |

## Guards

A guard is a flat function that returns context additions:

```ts
const auth = guard(async (ctx) => {
  const token = ctx.headers.authorization?.replace("Bearer ", "")
  const user = await verifyToken(token)
  if (!user) throw new KatmanError("UNAUTHORIZED")
  return { user, permissions: user.permissions } // [!code highlight]
})
```

Use in a procedure:

```ts
const me = query({
  use: [auth],
  resolve: ({ ctx }) => ctx.user, // ← typed as User [!code highlight]
})
```

<Callout type="info">
Guards that return `void` act as pure validators — they throw on failure but don't add context.
</Callout>

```ts
const adminOnly = guard((ctx) => {
  if (!ctx.permissions?.includes("admin"))
    throw new KatmanError("FORBIDDEN")
  // no return — nothing added to ctx
})
```

## Wraps

A wrap runs code **before** and **after** the handler:

```ts
const timing = wrap(async (ctx, next) => {
  const start = performance.now()
  const result = await next() // [!code highlight]
  logger.info({ ms: performance.now() - start })
  return result
})

const errorCapture = wrap(async (ctx, next) => {
  try {
    return await next()
  } catch (error) {
    Sentry.captureException(error)
    throw error
  }
})
```

## Stacking

```ts
const proc = mutation({
  use: [auth, rateLimit, adminOnly, timing, errorCapture],
  //    ───── guards ─────────────  ──── wraps ────────
  resolve: ({ ctx, input }) => { ... },
})
```

Execution:

```
→ auth         (guard)   ctx += { user, permissions }
→ rateLimit    (guard)   ctx += { rateLimit }
→ adminOnly    (guard)   throws if not admin
→ timing       (wrap)    start timer
  → errorCapture (wrap)  try/catch
    → resolve()          your handler
  ← errorCapture         catch errors
← timing                 log duration
```
