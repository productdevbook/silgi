title: SolidStart
description: Use Silgi with SolidStart API routes.
---

[SolidStart](https://start.solidjs.com) uses file-based API routes powered by Nitro. Silgi integrates via a catch-all endpoint.

## Setup

Create a catch-all API route:

```ts twoslash
// @noErrors
// src/routes/api/[...path].ts
import { createHandler } from 'silgi/solidstart'
import { appRouter } from '~/server/rpc'

const handler = createHandler(appRouter, {
  context: (event) => ({
    db: getDB(),
    headers: Object.fromEntries(event.request.headers),
  }),
})

export const GET = handler
export const POST = handler
```

Procedures are accessible at:

```
POST /api/users/list
POST /api/users/create
GET  /api/health
```

## Context from SolidStart middleware

Access middleware data through the event object:

```ts twoslash
// @noErrors
createHandler(appRouter, {
  context: (event) => ({
    user: event.locals.user,
  }),
})
```

## Analytics and Scalar

Enable the built-in [analytics dashboard](/docs/analytics) or [Scalar API reference](/docs/scalar) directly in the adapter:

```ts twoslash
// @noErrors
const handler = createHandler(appRouter, {
  context: (event) => ({ user: event.locals.user }),
  analytics: {
    auth: process.env.ANALYTICS_TOKEN,
  },
  scalar: true,
})
```
