## Quick Start

Convert your router into AI-callable tools:

```ts
import { generateText } from "ai"
import { openai } from "@ai-sdk/openai"
import { routerToTools } from "katman/ai"

const tools = routerToTools(appRouter) // [!code highlight]

const { text } = await generateText({
  model: openai("gpt-4o"),
  tools,
  prompt: "List all users, then create one named Bob",
})
```

The LLM sees your procedures as functions it can call — input schemas are automatically extracted from Zod.

## Single Procedure

```ts
import { procedureToTool } from "katman/ai"

const searchTool = procedureToTool("search", appRouter.search, {
  description: "Search the database for items matching a query",
})
```

## Filtering & Descriptions

```ts
const tools = routerToTools(appRouter, {
  filter: (path) => !path.startsWith("admin_"),
  descriptions: {
    users_list: "List all users in the system",
    users_create: "Create a new user account",
  },
})
```

<Callout>
Nested routers are flattened with underscores: `users.list` becomes `users_list` as a tool name.
</Callout>
