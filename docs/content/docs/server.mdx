## serve()

One-line Node.js server with batteries included:

```ts title="server.ts"
k.serve(appRouter, {
  port: 3000,          // auto-finds if busy (3000–3100)
  hostname: "0.0.0.0",
  scalar: true,        // API docs at /reference
  ws: true,            // WebSocket RPC on same port
})
```

All options:

```ts
k.serve(appRouter, {
  port: 3000,
  hostname: "127.0.0.1",
  scalar: {
    title: "My API",
    version: "2.0.0",
    description: "Production API",
    security: { type: "http", scheme: "bearer", bearerFormat: "JWT" },
    servers: [{ url: "https://api.example.com" }],
    contact: { email: "dev@example.com" },
  },
  ws: true,
  http2: { cert: "./cert.pem", key: "./key.pem" },
})
```

| Option | Description |
|---|---|
| `port` | Port number. Auto-finds next available if busy. |
| `hostname` | Bind address. Default `"127.0.0.1"`. |
| `scalar` | `true` or config object. Serves OpenAPI 3.1.0 + Scalar UI. |
| `ws` | `true` to enable WebSocket RPC on same port. |
| `http2` | `{ cert, key }` for HTTP/2 with TLS (HTTP/1.1 fallback). |

## handler()

Fetch API handler — works everywhere:

<Tabs items={['Node.js', 'Bun', 'Deno', 'Cloudflare Workers']}>
  <Tab value="Node.js">
  ```ts
  import { createServer } from "node:http"
  const handle = k.handler(appRouter)
  // Use with any Node.js HTTP adapter
  ```
  </Tab>
  <Tab value="Bun">
  ```ts
  const handle = k.handler(appRouter)
  Bun.serve({ fetch: handle, port: 3000 })
  ```
  </Tab>
  <Tab value="Deno">
  ```ts
  const handle = k.handler(appRouter)
  Deno.serve({ port: 3000 }, handle)
  ```
  </Tab>
  <Tab value="Cloudflare Workers">
  ```ts
  const handle = k.handler(appRouter)
  export default { fetch: handle }
  ```
  </Tab>
</Tabs>

## Content Negotiation

The server automatically responds in the format the client requests:

| Accept Header | Format | Use Case |
|---|---|---|
| `application/json` | JSON | Default |
| `application/x-msgpack` | MessagePack | 30% smaller payloads |
| `application/x-devalue+json` | devalue | Date, Map, Set, BigInt |

<Callout>
Content negotiation is automatic. Send the `Accept` header and the server responds accordingly. No configuration needed.
</Callout>

## Scalar API Reference

When `scalar` is enabled:

- **`/reference`** — Interactive API documentation UI
- **`/openapi.json`** — OpenAPI 3.1.0 JSON specification

The spec is generated at startup from your router — zero runtime cost per request.
