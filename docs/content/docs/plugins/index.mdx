Katman plugins follow the v2 pattern — guards for request-level logic, wraps for lifecycle, hooks for global events. No special plugin API. Just middleware.

<Callout>
All plugins are tree-shakeable. Import only what you use — unused plugins add zero bytes to your bundle.
</Callout>
