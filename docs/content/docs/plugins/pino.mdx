## Setup

```ts twoslash
// @noErrors
import pino from "pino"

const k = katman({
  context: (req) => ({}),
  hooks: loggingHooks({ logger: pino() }), // [!code highlight]
})
```

That's it. Every request, response, and error is logged.

## What Gets Logged

**Request received:**
```json
{ "level": "info", "path": "users/list", "hasInput": true, "msg": "request received" }
```

**Response sent:**
```json
{ "level": "info", "path": "users/list", "durationMs": 0.42, "msg": "response sent" }
```

**Error:**
```json
{ "level": "error", "path": "users/create", "error": "CONFLICT", "msg": "request error" }
```

## Options

```ts twoslash
// @noErrors
loggingHooks({
  logger: pino(),
  logRequests: true,   // default: true
  logResponses: true,  // default: true
})
```

Set `logRequests: false` to only log responses and errors.

<Callout>
The logger interface is Pino-compatible but not Pino-specific. Any logger with `info()`, `error()`, `warn()`, `debug()`, and `child()` methods works.
</Callout>
