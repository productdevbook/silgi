## Usage

```ts
import Fastify from "fastify"
import { katmanFastify } from "katman/fastify"

const app = Fastify()

// Existing REST routes
app.get("/legacy", async () => ({ type: "rest" }))

// Katman RPC routes under /rpc
app.register(katmanFastify(appRouter, {
  context: (req) => ({ user: req.user }),
}), { prefix: "/rpc" }) // [!code highlight]

app.listen({ port: 3000 })
```

Each procedure becomes a POST route:

```
POST /rpc/health
POST /rpc/users/list
POST /rpc/users/create
```

<Callout>
Fastify's built-in JSON parsing is used — no double-parse overhead. Content negotiation (JSON, MessagePack, devalue) works automatically.
</Callout>

## Options

| Option | Description |
|---|---|
| `context` | Context factory — receives Fastify request object |
| `prefix` | Via Fastify's `register({ prefix })` — standard Fastify pattern |
