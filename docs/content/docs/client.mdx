## Setup

<Steps>
<Step>

### Create a link

The link is the transport layer — how requests are sent to the server.

<Tabs items={['ofetch (recommended)', 'native fetch']}>
  <Tab value="ofetch (recommended)">
  ```ts
  import { createLink } from "katman/client/ofetch"

  const link = createLink({
    url: "http://localhost:3000",
    timeout: 5000,
    retry: 2,
  })
  ```
  </Tab>
  <Tab value="native fetch">
  ```ts
  import { RPCLink } from "katman/client/fetch"

  const link = new RPCLink({ url: "http://localhost:3000" })
  ```
  </Tab>
</Tabs>

</Step>
<Step>

### Create the client

```ts
import { createClient } from "katman/client"
import type { InferClient } from "katman"
import type { AppRouter } from "./router"

const client = createClient<InferClient<AppRouter>>(link)
```

</Step>
<Step>

### Use it

```ts
const users = await client.users.list({ limit: 10 })
const user = await client.users.create({ name: "Alice", email: "a@test.com" })
const result = await client.admin.settings.update({ theme: "dark" })
```

</Step>
</Steps>

## Link Options

```ts
const link = createLink({
  url: "http://localhost:3000",

  // Auth
  headers: () => ({
    authorization: `Bearer ${getToken()}`,
  }),

  // Reliability
  timeout: 5000,
  retry: 2,
  retryDelay: (ctx) => Math.min(1000 * 2 ** ctx.retryCount, 10000),

  // Binary protocol (30% smaller payloads)
  binary: true, // [!code highlight]

  // Interceptors
  onRequest: ({ options }) => { /* before send */ },
  onResponse: ({ response }) => { /* after receive */ },
  onRequestError: ({ error }) => { /* network failure */ },
})
```

## Binary Mode

Send `binary: true` to use MessagePack — 30% smaller payloads, native Date support:

```ts
const link = createLink({
  url: "http://localhost:3000",
  binary: true, // [!code highlight]
})
```

<Callout>
Binary mode is transparent to your code. The client encodes with MessagePack, the server detects the content type and responds in kind.
</Callout>

## Plugins

Plugins wrap the link to add behavior:

```ts
import { withRetry, withDedupe, withCSRF } from "katman/client/plugins"

const link = createLink({ url: "http://localhost:3000" })

const enhanced = withCSRF(
  withDedupe(
    withRetry(link, { maxRetries: 3, retryDelay: 1000 })
  ),
  { headerName: "x-csrf-token", headerValue: "abc123" }
)

const client = createClient<InferClient<AppRouter>>(enhanced)
```

| Plugin | What it does |
|---|---|
| `withRetry` | Retry failed requests with backoff |
| `withDedupe` | Collapse identical concurrent requests into one |
| `withCSRF` | Inject CSRF token header |
| `BatchLink` | Batch multiple calls into one HTTP request |
