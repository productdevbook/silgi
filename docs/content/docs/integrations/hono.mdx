title: Hono

[Hono](https://hono.dev) is a fast web framework that runs everywhere. Silgi's Hono adapter mounts your procedures as a Hono handler.

## Setup

```ts twoslash
// @noErrors
import { Hono } from 'hono'
import { silgiHono } from 'silgi/hono'

const app = new Hono()

// Existing routes
app.get('/api/health', (c) => c.json({ status: 'ok' }))

// Silgi RPC
app.all(
  '/rpc/*',
  silgiHono(appRouter, {
    context: (c) => ({
      db: getDB(),
      user: c.get('user'),
    }),
  }),
)

export default app
```

## Context from Hono

The `context` function receives Hono's `Context` object. Access middleware data via `c.get()`:

```ts twoslash
// @noErrors
silgiHono(appRouter, {
  context: (c) => ({
    user: c.get('user'), // from Hono middleware
    requestId: c.get('requestId'),
    env: c.env, // Cloudflare Workers bindings
  }),
})
```

## Cloudflare Workers

```ts twoslash
// @noErrors
import { Hono } from 'hono'
import { silgiHono } from 'silgi/hono'

const app = new Hono()
app.all(
  '/rpc/*',
  silgiHono(appRouter, {
    context: (c) => ({ db: c.env.DB }),
  }),
)

export default app
```
