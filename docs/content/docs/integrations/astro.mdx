---
title: Astro
description: Use Katman with Astro API routes.
---

[Astro](https://astro.build) supports server-side API routes via file-based routing. Katman integrates via a catch-all endpoint.

## Setup

Create a catch-all API route:

```ts twoslash
// @noErrors
// src/pages/api/rpc/[...path].ts
import { katmanAstro } from 'katman/astro'
import { appRouter } from '~/server/rpc'

const handler = katmanAstro(appRouter, {
  prefix: '/api/rpc',
  context: (ctx) => ({
    db: getDB(),
    headers: Object.fromEntries(ctx.request.headers),
  }),
})

export const GET = handler
export const POST = handler
```

Procedures are accessible at:

```
POST /api/rpc/users/list
POST /api/rpc/users/create
GET  /api/rpc/health
```

## SSR mode

Katman requires server-side rendering. Enable it in your Astro config:

```ts twoslash
// @noErrors
// astro.config.mjs
export default defineConfig({
  output: 'server',
})
```
