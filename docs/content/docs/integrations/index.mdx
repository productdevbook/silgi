## Which integration do I need?

| Scenario | Integration |
|---|---|
| You have an existing Fastify app and want to add RPC | [Fastify](/docs/integrations/fastify) |
| You're building a Next.js or TanStack Start app with server actions | [React](/docs/integrations/react) |
| You want to use React Query (or Vue/Solid/Svelte Query) for data fetching | [TanStack Query](/docs/integrations/tanstack-query) |
| You want LLMs to call your API procedures as tools | [AI SDK](/docs/integrations/ai-sdk) |

For standalone servers (no existing framework), you don't need any integration — just use `k.serve()` or `k.handler()` from the [Server](/docs/server) page.
